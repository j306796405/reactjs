# 用JSON-server模拟REST API的示例代码

将本项目 clone 或者下载到本地，进入 mock 目录后执行以下操作

````javascript
// 安装依赖包

npm install

// 运行mock
npm run mock

````

然后打开浏览器或使用 postman 访问 [http://localhost:3003](http://localhost:3003)
